# Plant Image Filename Mapping

Put your image files in this folder using the exact filenames below.

1. Red Yucca -> `red-yucca.jpg`
2. Lantana -> `lantana.jpg`
3. Lavender -> `lavender.jpg`
4. Texas Sage -> `texas-sage.jpg`
5. Bougainvillea -> `bougainvillea.jpg`
6. Oleander -> `oleander.jpg`
7. Agave -> `agave.jpg`
8. Aloe Vera -> `aloe-vera.jpg`
9. Pink Muhly Grass -> `pink-muhly-grass.jpg`
10. Deer Grass -> `deer-grass.jpg`
11. Rosemary -> `rosemary.jpg`
12. Ice Plant -> `ice-plant.jpg`
13. Blackfoot Daisy -> `blackfoot-daisy.jpg`
14. Bird of Paradise -> `bird-of-paradise.jpg`
15. Indian Hawthorn -> `indian-hawthorn.jpg`

Recommended image spec:
- Format: JPG
- Aspect ratio: 3:2
- Suggested size: 1200x800
- Suggested file size: under 800 KB each

If you prefer PNG, update each corresponding `image_url` extension in `backend/main.py`.
