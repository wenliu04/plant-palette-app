# Plant Palette & Search Tool

A tool to streamline plant search, HOA-based filtering, and palette generation for landscape design workflows.

## Version 1
- small plant dataset
- small HOA dataset
- GUI filters
- search results
- add/remove palette
- simple board preview

## Version 2.3
- bilingual UI labels (English / Chinese)
- language switcher on homepage

## Version 2.4
- floating AI assistant chat panel
- backend `/chat` endpoint with OpenAI integration
- tool-based plant recommendation with add-to-palette action
- backend agent logs (tool call + filter + latency)

## Version 2.5
- bloom timeline on Palette Board
- improved PDF workflow: preview + print/save as PDF + export adjustments
- image URL normalization/proxy path support for local PDF image embedding
